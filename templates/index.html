Twitter Trading Python Bot


This is a flask site created for displaying the Twitter bot that trades using Elon musk's tweets. It is created using the tweepy library and Text-Blob library to do live sentiment analysis of Elon's tweets and provide information about whether to sell, hold or buy stock of Tesla using sentiment of tweets' contents.

It is a very interesting setup, and we are going to see what trades it will make using this flask front-end. Now the more specific things are available in the different sections of the tabs. You can go to them one-by-one and explore the different aspects of this project and dive the analysis done by them.

We are also trying to integrate an API which can understand this information and make trades using Elon Musk's tweets using sites like ZeroDha and CoinBase Api.
